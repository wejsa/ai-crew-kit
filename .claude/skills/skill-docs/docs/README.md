## 키워드 매핑

개발 중 아래 키워드가 감지되면 해당 문서를 자동 참조합니다.

```
# 예시 (프로젝트에 맞게 수정)
결제, 승인, 인증 → payment-flow.md
정산, 수수료 → settlement.md
취소, 환불 → refund-cancel.md
API, 엔드포인트, 웹훅 → api-design.md
에러, 예외, 재시도 → error-handling.md
보안, 암호화, 인증 → security.md
```

## 문서 추가 방법

1. 이 디렉토리에 마크다운 파일 추가
2. 위 테이블에 문서 정보 추가
3. 키워드 매핑 섹션에 트리거 키워드 추가

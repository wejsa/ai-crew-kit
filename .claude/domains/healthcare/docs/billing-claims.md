# 보험 청구

## 개요

진료 완료 후 건강보험심사평가원(심평원)에 보험 청구하는 플로우를 정의합니다. 진료 예약/접수 플로우는 `appointment-flow.md`에서 다룹니다.

## 청구 상태 머신

```
created → submitted → adjudicated → settled
               ↘ rejected → resubmitted → adjudicated
                                  ↘ appealed → adjudicated
```

| 상태 | 설명 |
|------|------|
| created | 청구서 생성 (진료 완료 후) |
| submitted | 심평원 제출 완료 |
| adjudicated | 심사 완료 (인정/삭감/불인정) |
| settled | 지급 완료 |
| rejected | 청구 반려 (서류 미비, 코드 오류) |
| resubmitted | 보정 후 재제출 |
| appealed | 이의 신청 (삭감/불인정에 대한) |

### 허용 전이

| From | To | 조건 |
|------|----|------|
| created | submitted | 청구서 검증 완료 + 심평원 전송 |
| submitted | adjudicated | 심사 결과 수신 |
| submitted | rejected | 서류 미비/코드 오류 반려 |
| adjudicated | settled | 보험자(공단) 지급 |
| rejected | resubmitted | 오류 보정 후 재제출 |
| resubmitted | adjudicated | 재심사 결과 수신 |
| adjudicated | appealed | 삭감/불인정 항목 이의 신청 |
| appealed | adjudicated | 이의 심사 결과 |

## 청구 항목 구조

| 항목 | 설명 |
|------|------|
| 수가 코드 | 건강보험 수가 코드 (행위/치료재료/약제) |
| 진단 코드 | ICD-10 (KCD) 진단 코드 |
| 급여 구분 | 급여 / 비급여 / 100분의100 |
| 본인부담률 | 급여 항목의 환자 부담 비율 |
| 청구 금액 | 수가 × 횟수 × 일수 |

## 급여/비급여 구분

| 구분 | 설명 | 환자 부담 |
|------|------|----------|
| 급여 | 건강보험 적용 | 본인부담률 (20~60%) |
| 비급여 | 건강보험 미적용 | 전액 (100%) |
| 선별급여 | 일부 적용 | 본인부담률 50~90% |
| 100분의100 | 급여이나 전액 부담 | 전액 (100%) |

## 심사/삭감

| 유형 | 설명 |
|------|------|
| 인정 | 청구 전액 인정 |
| 삭감 | 일부 항목 불인정 (과잉 진료, 기준 초과) |
| 불인정 | 전체 불인정 (적응증 부적합 등) |

### 삭감 대응

| 규칙 | 설명 |
|------|------|
| 이의 신청 | 삭감 통보 후 90일 이내 |
| 소명 자료 | 진료기록, 검사결과, 의학적 근거 첨부 |
| 재심사 | 심평원 재심사 후 결과 통보 |

## 청구 검증 (제출 전)

| 검증 항목 | 설명 |
|----------|------|
| 수가 코드 유효성 | 폐지/변경된 수가 코드 사용 여부 |
| 진단-처치 적합성 | 진단 코드와 처치/약제의 급여 기준 적합 여부 |
| 중복 청구 | 동일 환자 동일 일자 중복 청구 여부 |
| 필수 항목 누락 | 청구에 필요한 필수 데이터 완전성 |

## 참고사항

- 청구 데이터 보존: 5년 (건강보험법)
- 청구 오류 발견 시 자진 취하 후 재청구 가능
- 비급여 항목은 심평원 청구 대상 아님 (환자 직접 수납)
- 진료 완료 → 청구 생성은 자동화 권장 (수가 자동 매핑)

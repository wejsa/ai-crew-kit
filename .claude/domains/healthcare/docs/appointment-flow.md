# 진료 예약 플로우

## 개요

환자의 진료 예약부터 접수, 진료, 수납까지의 임상 워크플로우를 정의합니다. 보험 청구는 `billing-claims.md`에서 별도로 다룹니다.

## 예약 상태 머신

```
scheduled → checked_in → in_progress → completed
  ↘ canceled
  ↘ no_show
```

| 상태 | 설명 |
|------|------|
| scheduled | 예약 확정 |
| checked_in | 접수 완료, 대기 중 |
| in_progress | 진료 진행 중 |
| completed | 진료 완료 |
| canceled | 예약 취소 |
| no_show | 미내원 (예약 시간 경과) |

### 허용 전이

| From | To | 조건 |
|------|----|------|
| scheduled | checked_in | 환자 내원 + 접수 |
| scheduled | canceled | 환자/병원 취소 (진료 전) |
| scheduled | no_show | 예약 시간 + 대기 시간 초과, 미내원 |
| checked_in | in_progress | 의사가 진료 시작 |
| checked_in | canceled | 접수 후 취소 (진료 전) |
| in_progress | completed | 의사가 진료 종결 |

### 예약 제약 조건

| 제약 | 설명 | 심각도 |
|------|------|--------|
| 시간 슬롯 중복 금지 | 동일 의사, 동일 시간대 2건 이상 예약 불가 | CRITICAL |
| 진료실 중복 금지 | 동일 진료실, 동일 시간대 2건 이상 예약 불가 | MAJOR |
| 장비 가용성 | 특수 장비 필요 시술은 장비 예약 확인 필수 | MAJOR |
| 최소 간격 | 연속 예약 간 최소 전환 시간 (기본 5분) | MINOR |

## 접수/수납 플로우

```
예약 확인 → 접수 (보험 확인) → 대기 → 진료 → 수납 (본인부담금)
                                              ↓
                                        처방전 발행 (외래 시)
```

### 접수 시 확인 사항

| 항목 | 설명 |
|------|------|
| 보험 자격 | 건강보험 자격 확인 (실시간 자격조회) |
| 본인 확인 | 신분증 또는 등록된 인증 수단 |
| 사전 동의 | 필요 시 동의서 서명 (시술/수술) |
| 이전 진료 | 의뢰/전원 시 의뢰서 확인 |

### 수납 항목

| 구분 | 설명 |
|------|------|
| 급여 | 건강보험 적용 항목 (본인부담률 적용) |
| 비급여 | 건강보험 미적용 항목 (전액 환자 부담) |
| 본인부담금 | 급여 항목의 환자 부담분 (통상 20~60%) |

## No-show 처리

| 규칙 | 설명 |
|------|------|
| 자동 전이 | 예약 시간 + 대기 시간(기본 30분) 경과 시 자동 no_show |
| 기록 | no_show 이력 기록 (향후 예약 시 참고) |
| 알림 | 예약 전 알림(전일, 2시간 전), no_show 후 재예약 안내 |
| 수수료 | 정책에 따라 no_show 수수료 부과 가능 |

## 참고사항

- 예약 변경은 cancel → 재예약 (수정 불가, 이력 추적)
- 진료 완료 후 보험 청구 플로우는 `billing-claims.md` 참조
- 응급 환자는 예약 없이 직접 checked_in → in_progress 가능
- 입원 환자는 별도 입퇴원 플로우 (admission → discharge) 적용

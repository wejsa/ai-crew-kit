# 처방 플로우

## 개요

전자 처방의 작성, 검증, 조제, 투약까지의 플로우를 정의합니다. 약물 상호작용 검증과 마약류 특별 규정을 포함합니다.

## 처방 상태 머신

```
draft → verified → dispensed → administered → completed
          ↘ rejected (약물 상호작용/용량 초과)
                      ↘ returned (미수령/반품)
verified → expired (유효기간 초과)
```

| 상태 | 설명 |
|------|------|
| draft | 의사가 처방 작성 중 |
| verified | 검증 완료 (DUR 통과, 용량 확인) |
| rejected | 검증 실패 (상호작용, 용량 초과, 알레르기) |
| dispensed | 약국에서 조제 완료 |
| returned | 환자 미수령 또는 반품 |
| administered | 투약 완료 (입원 환자) |
| completed | 처방 완료 (외래: 조제 후, 입원: 투약 후) |
| expired | 유효기간 초과 (일반 30일, 마약류 3일) |

### 허용 전이

| From | To | 조건 |
|------|----|------|
| draft | verified | DUR 통과 + 용량 검증 통과 |
| draft | rejected | DUR 실패 또는 용량 초과 (의사에게 반려) |
| verified | dispensed | 약사 조제 확인 |
| verified | expired | 유효기간 초과 (일반 30일, 마약류 3일) |
| dispensed | administered | 간호사 투약 확인 (입원 환자) |
| dispensed | returned | 환자 미수령 7일 초과 또는 반품 |
| dispensed | completed | 외래 환자 조제 완료 시점 |
| administered | completed | 투약 완료 기록 |

## 약물 상호작용 검증 (DUR)

| 검증 항목 | 설명 | 심각도 |
|----------|------|--------|
| 병용 금기 | 동시 투약 시 위험한 약물 조합 | CRITICAL (차단) |
| 상대 금기 | 주의 필요한 약물 조합 | WARNING (경고 후 허용) |
| 연령 금기 | 연령 부적합 약물 | CRITICAL (차단) |
| 임부 금기 | 임산부 투여 금지 약물 | CRITICAL (차단) |
| 용량 초과 | 체중/연령 기반 최대 용량 초과 | CRITICAL (차단) |
| 알레르기 | 환자 등록 알레르기 약물 | CRITICAL (차단) |
| 중복 투여 | 동일 성분 약물 중복 처방 | WARNING (경고 후 허용) |

### DUR 검증 플로우

```
처방 입력 → 환자 기존 투약 목록 조회 → 알레르기 목록 조회
  → DUR 엔진 검증 → 금기 발견 시 의사에게 경고
    → CRITICAL: 차단 (override 시 사유 기록)
    → WARNING: 경고 표시 후 의사 확인으로 진행
```

## 용량 검증

| 규칙 | 설명 |
|------|------|
| 체중 기반 | 소아: mg/kg 기준 최대 용량 |
| 연령 기반 | 고령자: 감량 필요 약물 확인 |
| 신장 기능 | eGFR 기반 용량 조절 (신독성 약물) |
| 간 기능 | 간대사 약물 용량 조절 |

## 마약류 처방 특별 규정

| 규칙 | 설명 | 근거 |
|------|------|------|
| 유효기간 3일 | 처방 후 3일 이내 조제 필수 | 마약류관리법 |
| 별도 기록 | 마약류 처방/투약 이력 별도 관리대장 | 마약류관리법 |
| 보고 의무 | 분기별 사용량 식약처 보고 | 마약류관리법 |
| 잔량 관리 | 미사용 마약류 반납/폐기 기록 | 마약류관리법 |

## 참고사항

- DUR 검증은 처방 저장(draft→verified) 시점에 자동 실행
- DUR CRITICAL 항목을 의사가 override 시 사유 코드 + 감사 로그 필수
- 처방 변경(용량, 약물) 시 기존 처방 무효화(rejected 전이) → 신규 처방 생성 (수정 금지)
- 처방 이력은 진료기록과 동일하게 보존 (의료법: 2년, 진료기록 연계 시 10년)

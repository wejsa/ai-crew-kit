# 헬스케어 도메인 용어집

| 용어 | 영문 | 설명 |
|------|------|------|
| PHI | Protected Health Information | HIPAA에서 정의한 보호대상 건강정보 (18개 식별자 포함) |
| EMR | Electronic Medical Record | 전자의무기록 — 단일 의료기관 내 진료 기록 |
| EHR | Electronic Health Record | 전자건강기록 — 여러 기관 간 공유 가능한 통합 건강 기록 |
| FHIR | Fast Healthcare Interoperability Resources | HL7 국제 의료정보 교환 표준 (R4 기준) |
| HL7 | Health Level 7 | 의료정보 시스템 간 데이터 교환 표준 기구/규격 |
| DUR | Drug Utilization Review | 의약품 적정 사용 검토 — 병용 금기, 용량 초과 등 자동 검증 |
| MRN | Medical Record Number | 의무기록번호 — 환자 식별 고유 번호 |
| ICD-10 | International Classification of Diseases | 국제질병분류 10차 개정판 (한국: KCD) |
| KCD | Korean Standard Classification of Diseases | 한국표준질병사인분류 |
| BTG | Break-the-Glass | 응급 시 환자-의료진 관계 없이 PHI에 접근하는 긴급 프로토콜 |
| BAA | Business Associate Agreement | HIPAA 요구 PHI 처리 위탁 계약 |
| IRB | Institutional Review Board | 임상연구윤리위원회 — 연구 참여 동의/데이터 사용 승인 기관 |
| HIPAA | Health Insurance Portability and Accountability Act | 미국 건강보험 이동성 및 책임에 관한 법률 |
| 비식별화 | De-identification | 개인 식별 불가하도록 데이터 가공 (Safe Harbor: 18개 식별자 제거) |
| 가명처리 | Pseudonymization | 식별자를 가명으로 대체, 매핑 테이블 별도 보관 |
| 수가 | Medical Fee | 의료 행위에 대한 보험 청구 금액 단위 |
| 급여 | Covered Service | 건강보험 적용 의료 서비스 (본인부담 20~60%) |
| 비급여 | Non-covered Service | 건강보험 미적용 — 환자 100% 부담 |
| 심평원 | HIRA (Health Insurance Review & Assessment Service) | 건강보험심사평가원 — 보험 청구 심사 기관 |
| 처방전 | Prescription | 의사가 발행하는 의약품 처방 문서 (유효기간: 일반 30일, 마약류 3일) |
| 조제 | Dispensing | 약사가 처방전에 따라 약을 준비하여 제공하는 행위 |
| 전원 | Transfer | 환자를 다른 의료기관으로 이송하는 절차 |
| 접수 | Registration/Check-in | 환자 내원 시 본인 확인 + 보험 자격 확인 절차 |

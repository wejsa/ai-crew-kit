# 컴플라이언스 체크리스트

이커머스 서비스 규정 준수 체크리스트입니다.

## 전자상거래법

| 항목 | 설명 | 심각도 |
|------|------|--------|
| 사업자 정보 표시 | 상호, 대표자, 사업자등록번호, 주소 | CRITICAL |
| 청약철회 | 7일 이내 청약철회 가능 안내 | CRITICAL |
| 환불 규정 | 환불 절차, 기간 명시 | CRITICAL |
| 배송 정보 | 배송 소요 시간, 배송비 명시 | MAJOR |
| 가격 표시 | 부가세 포함 가격 표시 | MAJOR |
| 결제 정보 | 결제 수단, 수수료 명시 | MAJOR |

### 필수 표시 정보

```
1. 상호 및 대표자명
2. 사업자등록번호
3. 통신판매업 신고번호
4. 사업장 주소
5. 고객센터 연락처
6. 개인정보관리책임자
```

## 소비자보호법

| 항목 | 설명 | 심각도 |
|------|------|--------|
| 청약철회 | 단순 변심 7일 이내 환불 | CRITICAL |
| 품질 하자 | 하자 발견 시 30일 이내 환불/교환 | CRITICAL |
| 과대광고 금지 | 허위/과장 광고 금지 | CRITICAL |
| 가격 비교 | 비교 가격은 실제 판매 가격 기준 | MAJOR |
| 후기 관리 | 조작/허위 후기 금지 | MAJOR |

### 청약철회 제한 사항

| 제한 사유 | 처리 |
|----------|------|
| 고객 과실 훼손 | 환불 불가 안내 |
| 시간 경과 가치 감소 | 사전 고지 필수 |
| 주문 제작 상품 | 사전 동의 필수 |
| 복제 가능 상품 (디지털) | 사전 고지 필수 |

|------|------|--------|

| 보유 기간 | 보유 기간 명시 및 준수 | CRITICAL |

| 위탁 고지 | 배송, 결제 등 위탁 업체 고지 | MAJOR |
| 파기 절차 | 목적 달성 후 지체 없이 파기 | MAJOR |

| 구분 | 항목 | 보유 기간 |
|------|------|----------|

| 선택 | 생년월일, 성별, 관심사 | 동의 철회 시 즉시 |

법적 요구사항 미준수 시 CRITICAL로 즉시 수정이 필요합니다.

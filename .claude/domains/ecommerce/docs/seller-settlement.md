# 셀러 정산

## 개요

마켓플레이스에서 셀러에게 판매 대금을 정산하는 프로세스입니다. 구매 확정 후 정산 주기(D+N)에 따라 수수료를 차감하고 셀러 계좌로 이체합니다.

## 정산 상태 머신

```
pending → calculated → confirmed → transferred → completed
                          ↘ disputed → resolved → confirmed
                                                 ↘ adjusted → confirmed
```

| 상태 | 설명 | 전이 조건 |
|------|------|----------|
| pending | 구매 확정 대기 | 서브오더 confirmed 시 정산 대기열 진입 |
| calculated | 정산액 산출 완료 | 정산 배치 실행 (D+N 도래) |
| confirmed | 정산 확정 | 셀러 이의 없음 또는 이의 해결 |
| disputed | 셀러 이의 제기 | 셀러가 정산 내역에 이의 |
| resolved | 이의 해결 | 운영팀 검토 완료 |
| adjusted | 정산액 조정 | 이의 인정 시 재산출 |
| transferred | 이체 완료 | 지급 배치 실행 |
| completed | 정산 완료 | 이체 확인 + 세금계산서 발행 |

### 허용 전이

| From | To | 조건 |
|------|----|------|
| pending | calculated | 정산 배치 실행 (D+N 도래) |
| calculated | confirmed | 셀러 이의 없음 (이의 제기 기한 경과) |
| confirmed | transferred | 지급 배치 실행 |
| transferred | completed | 이체 확인 + 세금계산서 발행 |
| confirmed | disputed | 셀러 이의 제기 |
| disputed | resolved | 운영팀 검토 완료 (이의 기각) |
| disputed | adjusted | 운영팀 검토 완료 (이의 인정, 재산출) |
| resolved | confirmed | 기각 확정 → 원래 정산액 유지 |
| adjusted | confirmed | 재산출 완료 → 조정된 정산액으로 확정 |

## 정산 주기

| 주기 | 설명 | 적용 대상 |
|------|------|----------|
| D+7 | 구매 확정 후 7영업일 | 신규/일반 셀러 |
| D+3 | 구매 확정 후 3영업일 | 우수 셀러 |
| D+1 | 구매 확정 후 1영업일 | 프리미엄 셀러 |
| 즉시 | 구매 확정 즉시 | 특약 셀러 (별도 계약) |

## 정산 계산 구조

```
총 판매액 (서브오더 합계)
  - 플랫폼 수수료 (카테고리별 × 등급 할인)
  - 결제 수수료 (PG 수수료 분담분)
  - 프로모션 분담금 (플랫폼 주도 할인의 셀러 부담분, 프로모션 계약에 따라 0~100%)
  + 배송비 수취분 (구매자 부담 배송비 중 셀러 몫)
  - 반품/환불 차감분 (정산 기간 내 발생한 환불)
  = 셀러 정산액
```

### 계산 규칙

| 규칙 | 설명 |
|------|------|
| BigDecimal 필수 | 모든 금액은 BigDecimal, HALF_UP 반올림 |
| 음수 정산 방지 | 정산액이 0 미만이면 다음 주기로 이월 |
| 통화 단위 | 원(KRW) 기준, 소수점 없음 (정수 원 단위) |
| 세금계산서 | 정산 완료 시 전자 세금계산서 자동 발행 |

## 반품/환불과 정산

| 시점 | 처리 |
|------|------|
| 정산 전 반품 | pending 상태에서 차감, 정산 대상에서 제외 |
| 정산 후 반품 | 다음 정산 주기에서 차감 (역정산) |
| 분쟁 중 반품 | 분쟁 해결 시까지 해당 금액 홀드 |

## 정산 리포트

셀러에게 제공하는 정산 명세:

| 항목 | 설명 |
|------|------|
| 정산 기간 | 시작일 ~ 종료일 |
| 총 판매 건수/금액 | 서브오더 단위 집계 |
| 수수료 내역 | 카테고리별 수수료 상세 |
| 차감 내역 | 반품/환불/프로모션 분담금 |
| 최종 정산액 | 이체 금액 |
| 이체 정보 | 셀러 계좌, 이체일 |

## 참고사항

- 정산 배치는 매일 자정 실행 (주기 도래 건 일괄 처리)
- 정산 데이터 보존 기간: 5년 (세법 기준)
- 셀러 정산 이의 제기 기한: 정산 확정 후 7일 이내
- 마켓플레이스 구조는 `marketplace.md` 참조

# 컴플라이언스 체크리스트

금융 서비스 규정 준수 체크리스트입니다.

## PCI-DSS 준수

| 항목 | 설명 | 심각도 |
|------|------|--------|
| 카드번호 저장 | PAN은 암호화하여 저장 (AES-256) | CRITICAL |
| CVV/CVC 저장 금지 | 인증 후 즉시 삭제, 저장 절대 금지 | CRITICAL |
| PIN 저장 금지 | PIN 데이터 저장 절대 금지 | CRITICAL |
| 카드번호 마스킹 | 화면/로그 표시 시 마스킹 (앞6자리 + **** + 뒤4자리) | CRITICAL |
| 카드번호 로깅 금지 | 평문 카드번호 로그 출력 금지 | CRITICAL |
| 암호화 키 관리 | 키 분리 보관, 주기적 교체 | MAJOR |
| 접근 제어 | 카드 데이터 접근 최소 권한 원칙 | MAJOR |

## 전자금융감독규정

| 항목 | 설명 | 심각도 |
|------|------|--------|
| 거래 로그 보존 | 5년 이상 보존 | CRITICAL |
| 감사 추적 | 모든 거래 변경 이력 기록 | CRITICAL |
| 접근 통제 | 비인가 접근 차단 | CRITICAL |
| 암호화 전송 | TLS 1.2 이상 필수 | CRITICAL |
| 이중 인증 | 고위험 거래 시 2FA 적용 | MAJOR |
| 이상 거래 탐지 | FDS 연동 또는 자체 탐지 (전자금융감독규정 제17조) | CRITICAL |
| 백업 정책 | 일일 백업, 복구 테스트 | MAJOR |

## 개인정보보호법

| 항목 | 설명 | 심각도 |
|------|------|--------|
| 수집 동의 | 개인정보 수집 시 명시적 동의 | CRITICAL |
| 목적 명시 | 수집 목적 명확히 고지 | CRITICAL |
| 최소 수집 | 필요 최소한의 정보만 수집 | MAJOR |
| 보유 기간 | 목적 달성 후 지체 없이 파기 | MAJOR |
| 제3자 제공 | 제공 시 별도 동의 획득 | CRITICAL |
| 열람/정정/삭제 | 정보주체 권리 보장 | MAJOR |
| 개인정보 암호화 | 전송/저장 시 암호화 | MAJOR |

## 자금세탁방지 (AML)

| 항목 | 설명 | 심각도 |
|------|------|--------|
| 고객 확인 (KYC) | 신원 확인 절차 | MAJOR |
| 의심 거래 보고 | 이상 거래 보고 체계 | MAJOR |
| 거래 한도 | 일/월 거래 한도 설정 | MINOR |

## 감사 로그 필수 항목

| 필드 | 설명 | 필수 |
|------|------|------|
| timestamp | 이벤트 발생 시각 | ✅ |
| userId | 수행 사용자 | ✅ |
| action | 수행 동작 | ✅ |
| resourceType | 대상 리소스 유형 | ✅ |
| resourceId | 대상 리소스 ID | ✅ |
| ipAddress | 요청 IP | ✅ |
| result | 성공/실패 | ✅ |
| detail | 상세 정보 (마스킹) | ⚠️ |

## 오픈뱅킹 규정

| 항목 | 설명 | 심각도 |
|------|------|--------|
| API 인증서 관리 | 금융결제원 발급 인증서 안전 보관, 만료 전 갱신 | CRITICAL |
| 거래고유번호 멱등성 | 모든 이체 요청에 고유번호 부여, 중복 요청 방지 | CRITICAL |
| 토큰 암호화 저장 | access_token/refresh_token AES-256 암호화 저장 | CRITICAL |
| 토큰 로깅 금지 | 오픈뱅킹 토큰 평문 로그 출력 금지 | CRITICAL |
| 이체 한도 검증 | 1회/1일/1월 이체 한도 초과 여부 사전 확인 | MAJOR |
| 수취인 확인 | 이체 전 수취인 조회 API로 예금주 검증 | MAJOR |
| 타임아웃 준수 | 잔액/수취인 조회 3초, 거래내역/이체 5초 타임아웃 설정 | MAJOR |

## 마이데이터 규정

| 항목 | 설명 | 심각도 |
|------|------|--------|
| 개별 동의 | 정보제공자별·항목별 개별 동의 획득 | CRITICAL |
| 동의 철회 처리 | 철회 요청 시 즉시 전송 중단 + 5영업일 이내 데이터 삭제 | CRITICAL |
| 데이터 암호화 | 수집 데이터 AES-256 암호화 저장 | CRITICAL |
| 동의 유효기간 | 최대 1년, 만료 시 자동 전송 중단 | MAJOR |
| 폐기 로그 | 데이터 삭제 시 폐기 증명 기록 | MAJOR |
| 분리 보관 | 정보주체별 데이터 격리 저장 | MAJOR |
| 동의 이력 보존 | 동의/철회 이력 5년 보존 (감사 추적) | MAJOR |
| 마이데이터 토큰 로깅 금지 | access_token/refresh_token 평문 로그 출력 금지 | CRITICAL |

## 사용 방법

이 체크리스트는 `skill-review`, `skill-review-pr` 실행 시 자동으로 적용됩니다.
CRITICAL 항목 미준수 시 리뷰 통과 불가합니다.
